# Decision Log Template

Registrar decisiones humanas que los agentes deben respetar.

## Formato

| Fecha | Decisión | Razón | Impacto | Reversible | Quién | Agentes afectados |
|-------|----------|-------|---------|------------|-------|-------------------|
| | | | | | | |

## Campos

- **Fecha:** Cuándo se tomó la decisión.
- **Decisión:** Qué se decidió, en una frase.
- **Razón:** Por qué se decidió.
- **Impacto:** Qué cambia como resultado.
- **Reversible:** Sí/No. Si sí, qué se necesita para revertir.
- **Quién:** Quién tomó la decisión.
- **Agentes afectados:** Qué agentes deben respetar esta decisión.

## Reglas

- Toda decisión que afecte el comportamiento del agente debe registrarse acá.
- No registrar decisiones operativas triviales.
- Si una decisión contradice un perfil, actualizar el perfil también.
- Las decisiones registradas acá prevalecen sobre interpretaciones del agente.

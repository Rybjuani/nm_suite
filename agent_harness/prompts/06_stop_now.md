# Prompt: Stop Now

Emergencia. Cortar sesión inmediatamente.

## Instrucciones

1. **Cortar sesión.** No seguir trabajando.
2. **No editar más archivos.**
3. **No hacer commit.**
4. **No hacer push.**
5. **Entregar resumen mínimo.**

## Resumen mínimo

```
## SESIÓN CORTADA

### Motivo
(razón de la interrupción)

### Estado
- Archivos tocados: (lista)
- Último cambio: (descripción breve)
- Tests: (estado último conocido)

### Git status
(pegar git status -sb)

### Acción requerida
- El humano debe decidir: commit parcial / rollback / descartar
```

## Regla

> No argumentar. No continuar. No "sólo un cambio más". Cortar y entregar.
